Kode Kivy: Menggunakan PageLayout

1. Mengimpor Modul

from kivy.app import App
from kivy.uix.button import Button
from kivy.uix.pagelayout import PageLayout

- `App`: kelas utama aplikasi Kivy.
- `Button`: widget untuk membuat tombol.
- `PageLayout`: layout yang membagi tampilan menjadi beberapa halaman yang bisa digeser (swipe).

2. Membuat Kelas Aplikasi

class MyApp(App):

Membuat kelas `MyApp` yang merupakan turunan dari `App`. Ini adalah kelas utama aplikasi yang akan dijalankan.

3. Fungsi build() - Membangun Tampilan

    def build(self):
        p_layout = PageLayout()

- `build(self)`: Fungsi yang akan dipanggil saat aplikasi dijalankan.
- `p_layout`: objek `PageLayout` yang menjadi layout utama aplikasi.

4. Menambahkan Tombol ke Halaman PageLayout

        btn1 = Button(text='Rian', size_hint=(0.2, 0.2))
        btn2 = Button(text='Pangestu', size_hint=(0.2, 0.2))
        btn3 = Button(text='Ganteng', size_hint=(0.2, 0.2))

- `Button(...)`: Membuat tombol dengan teks sesuai nama.
- `size_hint=(0.2, 0.2)`: Menentukan ukuran tombol, 20% dari lebar dan tinggi layout.

5. Menambahkan Tombol ke dalam PageLayout

        p_layout.add_widget(btn1)
        p_layout.add_widget(btn2)
        p_layout.add_widget(btn3)

- Setiap `add_widget(...)` menambahkan tombol ke halaman PageLayout.
- Setiap tombol akan berada di halaman yang bisa digeser secara horizontal.

6. Mengembalikan Layout

        return p_layout

Mengembalikan `p_layout` sebagai tampilan utama dari aplikasi.

7. Menjalankan Aplikasi

MyApp().run()

- Membuat objek dari kelas `MyApp`.
- Memanggil `run()` untuk menjalankan aplikasi.
